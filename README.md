Bash Configuration
==================

Bash configuration files for my development environment.

Includes:
- bashrc
- bash aliases
- platform specific configuration
- completions

Assumptions:
- installed on a platform with bash
- installed into ~/.bash-config
- appropriate symlinks have been created for:
  - .bash_aliases
  - .bashrc
  - .bash_profile
  - .gitconfig
  - current-platform
